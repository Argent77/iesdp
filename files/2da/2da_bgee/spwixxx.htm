spwixxx.2da

This file is a resource file for a spell using effect 214 (select spell). When the main spell is cast, this file is used to present several further spell options.
The first column contains identifiers for the spell names.
The second column contains the filename of each spell.
The third column is unknown.

In BG2EE we have (spwi417):
